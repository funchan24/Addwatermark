## Addwatermark

*水印助手*

### 用法

* 双击 **start.bat** 启动程序

### 说明

* 给图片、PDF文件支持全屏水印
* 可以调整水印的字体、不透明度、旋转角度
* 支持将文字水印转为二维码水印
* 感谢[Python](https://www.python.org/)、[ttkbootstrap](https://github.com/israel-dryer/ttkbootstrap)等
